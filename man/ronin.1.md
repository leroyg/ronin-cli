# ronin 1 "April 2012" Ronin "User Manuals"

## SYNOPSIS

`ronin` [*COMMAND*] [*options*]

## DESCRIPTION

Runs a Ronin COMMAND or the starts Ronin Console.

## ARGUMENTS

*COMMAND*
  The Ronin command to execute.

## OPTIONS

Additional options for the COMMAND or the Ronin Console.
